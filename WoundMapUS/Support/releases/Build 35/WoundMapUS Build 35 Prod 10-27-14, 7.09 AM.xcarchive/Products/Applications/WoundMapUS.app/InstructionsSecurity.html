Patient information needs to be secure. WoundMap requires a user to sign in using a unique username and password. To protect patient privacy, you should sign out when finished using the app.

iOS Security Features

In addition to the WoundMap username/password, you can elect to use iOS security features. The most obvious iOS security feature is the iOS Pin Code that provides access to the device. Other security features are detailed in Apple's web site at http://www.apple.com/iphone/business/it/security.html.

Note that you may elect to not enter the patient's name when adding a new patient. Use the EMR/Record Identifier to identify the patient.

For more detailed instructions see the WoundMap website at www.theWoundMap.com
